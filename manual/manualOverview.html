Manual Overview

This section provides extensive documentation on the use of WebTest.

Contents

Samples
An introduction to webtest. This page show the stepwise development of a first test suite.
Properties
Introduction to webtest property types.
Syntax Reference - Core Steps
Reference pages for the steps. The core steps are all the basic steps of webetest.
Syntax Reference - Extension Steps
Reference pages for the extension steps. Extension steps are part of the distribution, but have been less heavily used and are less well documented than the Core steps and are also more likely to change.
Syntax Reference - Filter Steps
Reference pages for the filter steps. Filter steps allow the content of web responses to be manipulated (or filtered ) for the purposes of making the response content more suitable for testing purposes. Filter steps are part of the distribution, but have been less heavily used and are less well documented than the Core steps and are also more likely to change.
Syntax Reference - PDF Steps
Reference pages for the PDF steps that let webtest verify the content and properties of PDF documents.
Syntax Reference - Excel Steps
Reference pages for the Excel steps that let webtest verify the content and properties of XLS documents.
Syntax Reference - Email Steps
Reference pages for the Email steps that let webtest verify the content and properties of Email messages and Email attachments.
Step Index
An alphabetical index to all the steps.
Groovy Testing
Tutorial page to show how to use Canoo WebTest trough Groovy.
