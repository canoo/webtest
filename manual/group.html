WebTestRecorder
the Firefox Extension for WebTest

Core Step group

Description

This step allows grouping and giving a description to a sequence of nested steps. Any kind of step can be nested.

Parameters

description
Required? no
The description of this test step.

Nested Parameters

step
Required? yes
A webtest step.
